# KPress 0.2.2

KPress `0.2.2` is a compatible maintenance release for hosts that render and export
documents from storage abstractions rather than plain UTF-8 files.

## Host-Supplied Export Content

- `KPressExportRequest.source_text` accepts document text that a host has already
  decoded or decompressed.
- `path` remains the logical document identity and base directory for relative assets,
  so host-provided content preserves titles and sibling image resolution.
- Omitting `source_text` retains the existing behavior of reading `path` as UTF-8.

## Compatibility

The new request field is optional and does not change the typed asset-manifest schema,
renderer API, publisher output, or fragment hooks introduced in the `0.2` line.
Hosts pinned to `0.2.0` or `0.2.1` can upgrade directly.

## Verification
